# 赞助页面程序

## 项目概述
这是一个为个人主页配置文件（config.js）设计的赞助页面程序，包含独特的二次元风格界面，使用指定的图片资源创建了一个美观且功能完善的赞助页面。

## 功能特性

### 图片展示
- **背景图片**: `IMG_3787.JPG` - 作为页面背景，创建沉浸式视觉体验
- **展示图片**: 
  - `IMG_3814.JPG` - 在左侧面板中展示
  - `IMG_3815.JPG` - 在右侧面板中展示
- 两张展示图片并排平铺显示，便于对比和展示

### 二次元UI元素
- 彩色装饰角标（🎀 ⭐ 💖 🌸）- 为图片容器增添可爱的二次元风格
- 动态彩虹边框效果 - 图片容器周围有流动的彩虹渐变边框
- 卡平滑悬停动画 - 鼠标悬停时图片容器会轻微上浮并增强阴影效果
- 渐变色设计 - 多处使用渐变色增强视觉效果

### 响应式设计
- 页面适配各种屏幕尺寸
- 移动设备上自动调整为垂直布局
- 确保在不同设备上都有良好的用户体验

## 配置说明

### config.js集成
该赞助页面已集成到`config.js`文件中，具体配置如下：

```javascript
{
  go: "🗂️ 前往", 
  img: "/img/xbmt1.jpg", 
  title: "赞助", 
  subtitle: "春秋蝉鸣少年归", 
  text: "If you see this line, I've managed to get your attention.", 
  url: "sponsor.html", 
  show: false
}
```

## 使用方法
1. 确保所有图片文件都放置在与`sponsor.html`相同的目录中
2. 在`config.js`中确保赞助项目的URL指向`sponsor.html`
3. 部署到Web服务器后，点击主页上的"赞助"卡片即可访问该页面
